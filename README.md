### How to use this book

This repository is structured as a course, not a random list. It is designed to take you from "drawing lines" to "painting with logic."

**Phase I: Know The Grid (The Geometry)**
We ignore the numbers and focus entirely on the loops.
* **The Primary:** Learning the coordinate system ($i, j$) using squares and triangles.
* **The Derived:** How to glue simple shapes together to form Pyramids and Diamonds.
* *Goal:* To be able to look at any shape and mentally break it down into $i$ and $j$ loops.

**Phase II: Rook, Camel, and Horse (The Data)**
Once the shape is steady, we change what we print inside it.
* **The Rook (Straight Lines):** Row or Column based values (1, 2, 3...).
* **The Camel (Diagonals):** Logic that changes along the diagonal ($i == j$).
* **The Horse (Jumps):** Boolean logic (Evens, Odds, and alternators).

**Phase III: The Endgame (Complex Logic)**
The final test. Here we combine geometry and data to create alphabets, hollow patterns, and dynamic shapes.
